React has been leading the industry for several years now and it is one of the most popular Javascript libary. In this document, we'll learn about the basics of React and why do we need it.

### What is React?

React is a declarative and effecient Javascript libary for building user interfaces (UI). It allows developers to break down complex UI into small and isolated pieces of code called 'components'. Components can also be reused, which makes developing large scale applications easy.
